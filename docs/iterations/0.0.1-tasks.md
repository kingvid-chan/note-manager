### T005 — 认证 API

- `src/note_manager/schemas/auth.py`
- `src/note_manager/routers/auth.py`
- `src/note_manager/dependencies.py`（`get_current_user`）

**验收标准**:

## 阶段 2: 核心 API

### T006 — 笔记服务层

**涉及文件**:
- `src/note_manager/services/note_service.py`

**验收标准**:
- [ ] `create_note(db, user_id, title, content_md, tag_ids) -> Note`: 创建笔记 + 关联标签，写 ActivityLog
- [ ] `get_note(db, note_id, user_id) -> Note | None`: 所有者校验（非 owner 返回 None，由路由层转 403/404）
- [ ] `list_notes(db, user_id, page, size, q, tag) -> (items, total)`: 分页 + 标题/内容 LIKE 搜索 + 标签名筛选（AND 逻辑，多标签逗号分隔）
- [ ] `update_note(db, note_id, user_id, title, content_md, tag_ids) -> Note | None`: 更新笔记 + 同步标签关联，写 ActivityLog
- [ ] `delete_note(db, note_id, user_id) -> bool`: 软标记删除（无恢复需求时可真删除），写 ActivityLog
- [ ] 所有查询按 `user_id + created_at DESC` 排序

**预估复杂度**: L

**依赖**: T001, T002
